# Measurement of carbon emissions

## Code Carbon

See [https://github.com/mlco2/codecarbon](https://github.com/mlco2/codecarbon)

### Start monitoring

```cmd 
codecarbon config
```

``` cmd
cd codecarbon
codecarbon monitor
```

### Setup stress

Start *CPUSTRES.exe* from *SysInternals*
